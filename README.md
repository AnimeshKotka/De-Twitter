## Deployment

To deploy this project locally run

```bash
  yarn hardhat deploy 
```

To deploy this project on testnet run

```bash
  yarn hardhat deploy --network goreli
```
For client app run this
```bash
  cd client
```
Then
```bash
  npm i && npm start
```
## Running Tests

To run tests, run the following command

```bash
  yarn hardhat test
```
